Summary

I am really proud of how this site came out. I wanted to try creating a website that captures a simplistic design that is easy to navigate. Originally I was wondering how I would create a navigational system that made sense. I knew I wanted to have a table of contents for each page for in-page navigation and a top navigation bar for maneuvering through the other pages. I also have written that many papers in a while, so I wanted to practice writing again hence the large amount of content.

I feel that I really tackled the usage of media queries to create a responsive design for any device. I also made sure to keep the formatting of the pages consistent by first designing the HTML markup, creating the CSS, then revising the HTML markup again to make it easier to stylize. In addition, I tried to use flexbox to display content and it turned out well when resizing the window. The anchor links all have CSS rollover effects.

I could definitely see some improvement with the current version of the site. Originally I wanted to add JavaScript for the purpose of having scrollable code examples within the HTML. The table of contents bar could arguably be on the right side and not the left side. I was also not sure how to dynamically change the size of an embedded video, so I wrapped it in a flexbox container. With some of the images, I got lazy with and used floats to make the text wrap around the image.
